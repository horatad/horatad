⛔ บทสรุปที่แก้ไขแล้ว — MR×MR ถูกหักล้าง

  • MR (มฤตยู) ใน engine มีคาบ ~83 ปี — transit_MR × natal_MR aspects เป็น age proxy ทั้งหมด
  • MR conj → อายุ ~83, MR oppo → อายุ ~41.5 — ตรงกับ life stage ตามธรรมชาติ ไม่ใช่โหราศาสตร์
  • Skeptic Test 5 (age distribution): median 82.9 ปี ใน MR-conj deaths = ยืนยัน age confound
  • MR×MR rules ต้องถูก REMOVED จาก JULIAN valid findings ทั้งหมด

✅ สิ่งที่ยังคงอยู่ — SU×SA Polarity

  • transit_SU มีคาบ 1 ปี — ไม่เป็น age proxy (คนทุกอายุได้รับ SU aspects ทุกปี)
  • Hard aspects (SU→SA): death lift ~1.12-1.18 | Soft aspects: award lift ~1.12-1.16
  • Directional effect: SU→SA works, SA→SU ไม่ work — ไม่ใช่ artifact symmetric
  • Benjamini-Hochberg FDR: p_adj < 0.05 — ผ่าน multiple testing
  • Effect size เล็ก (1.1-1.2) — ต้องการ degree-level data + birth time precision สูงขึ้น
  • ข้อมูล Wikidata = บุคคลมีชื่อเสียง ไม่ใช่ประชากรทั่วไป
