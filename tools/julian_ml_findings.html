บทสรุป — สถานะหลักฐาน

  • MR×MR conjunction เป็นอันดับ 1 จาก 2,100 การรวมกัน ด้วย lift = 2.264
  • ผ่านการหักล้าง 3 วิธี: orbital period confound, generic MR transit, generic natal MR
  • Effect เฉพาะเจาะจง (specificity) — เกิดเฉพาะ MR×MR ไม่กระจาย → hard to explain as artifact
  • Benjamini-Hochberg FDR correction: p_adj ≈ 0 — ผ่าน multiple testing อย่างเข้มงวด
  • ข้อจำกัด: ข้อมูล Wikidata = บุคคลมีชื่อเสียง ไม่ใช่ประชากรทั่วไป
  • ข้อจำกัด: sign-level resolution (ไม่ใช่ degree-level) — precision ต่ำกว่าที่เหมาะสม
  • SU×SA เป็นการค้นพบรองที่น่าสนใจ แต่ lift ~1.1-1.2 ยังต้องการข้อมูลเพิ่ม
