## WHY LOG

- **manual loop ก่อน auto (2026-05-29)** — Phase 3 (Meta API auto-post) ต้อง page follower + App Review ซึ่งใช้เวลา ระหว่างนี้ทำ manual loop ให้สมบูรณ์ก่อน (copy → post → mark) แล้วค่อยต่อ pipe ไป API ทีหลัง — ไม่ block content cadence
- **posted/ feedback loop สำคัญ (2026-05-29)** — curator ใช้ `posted/` diversity check กันโพสต์ category ซ้ำ 7 วัน ถ้า posted/ ว่าง = check ไม่ทำงาน = อาจโพสต์ education ซ้ำกัน 7 วัน ปิด loop จึง unlock diversity
- **PAT = optional แต่สำคัญ (2026-05-29)** — ไม่ใส่ PAT ก็ copy ได้ แต่คิวไม่ย้าย = scheduled สะสม = posted/ ว่างซ้ำอีก เป็น feedback loop ที่ต้องปิดเพื่อให้ระบบทำงานถูกต้องในระยะยาว
