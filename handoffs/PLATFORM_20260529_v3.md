**วิธีที่ปลอดภัย (browser-only, secret ไม่เข้าแชท) — ค้างที่ขั้น 1:**
1. **เพิ่ม redirect URI** → App → Facebook Login → Settings → Valid OAuth Redirect URIs:
   `https://horatad.com/privacy.html`
2. **short token** (ข้าม Explorer, เติม App ID เอง, เปิดบนเบราว์เซอร์):
   ```
   https://www.facebook.com/v23.0/dialog/oauth?client_id=APP_ID&redirect_uri=https://horatad.com/privacy.html&scope=pages_show_list,pages_read_engagement,pages_manage_posts&response_type=token
   ```
   → login → ดู URL bar `#access_token=...` copy
3. **long-lived** (เติม App ID + Secret เองบน address bar):
   `https://graph.facebook.com/v23.0/oauth/access_token?grant_type=fb_exchange_token&client_id=APP_ID&client_secret=APP_SECRET&fb_exchange_token=SHORT`
4. **Page token** (ไม่หมดอายุ):
   `https://graph.facebook.com/v23.0/me/accounts?access_token=LONG_LIVED`
   → copy Page `id` + `access_token`
5. **debug ยืนยัน** Expires: Never → developers.facebook.com/tools/debug/accesstoken
6. **GitHub Secrets** → `FB_PAGE_ID`, `FB_PAGE_TOKEN`
7. **ทดสอบ** → Actions → FB Auto-post → Run → dry_run=true

> ⚠️ Claude ห้ามขอ App Secret มาในแชท — ให้ template ปีเตอร์เติมเองเท่านั้น (Secret Handling Rule)
