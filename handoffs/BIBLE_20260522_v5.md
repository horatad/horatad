### Rule Taxonomy (3 types)
```
Type 1: Natal Atomic
        ดาว X natal คุณภาพ Y → ลักษณะ/ศักยภาพ (คงที่ตลอดชีวิต)

Type 2: Natal Combination
        ดาว X natal + ดาว Y natal → ศักยภาพซ่อนอยู่ (คงที่)

Type 3: Transit × Natal
        transit pattern → natal pattern → เหตุการณ์ "เมื่อไหร่" (เปลี่ยนตามเวลา)
```


### 1. Color Mixing — rules ไม่ลบล้างกัน
