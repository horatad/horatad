**Strict parser validation layers (julian_astrotheme.mjs):**
1. Context-aware — ต้องเจอ "birth time" / "heure de naissance" / "born at" ก่อนค่อย match time
2. Cross-validate 3 formats (European h, AM/PM, 24h) — ต้องตรงกัน else reject
3. Blacklist defaults: 12:00, 00:00, 11:00, 12:30
4. Batch cluster detection: time ซ้ำ >3x ใน batch → drop (parser อาจจับ scraper clock)
5. Lat+lng must come together — partial coords → reject

**Strict SPARQL fictional filter (julian_config.mjs):**
