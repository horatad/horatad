## WHY LOG
- **iOS QR ใช้ opacity:0 แทน position:-9999px** — Safari ไม่ render canvas ที่อยู่นอก viewport จริงๆ แต่ render ได้ถ้าอยู่บนหน้าจอและซ่อนด้วย opacity แทน
- **Bundle qrcode.min.js ลง repo** — ไม่พึ่ง CDN ที่อาจล่ม ทำให้ capture offline ได้ทันที
- **ลบ section ปรัชญาออกจาก about page** — เนื้อหานั้นเป็น internal dev guideline ไม่ใช่ข้อมูลที่ user ต้องการอ่านในแอป
- **JULIAN ชี้ไปที่ GitHub raw แทน CF Release** — CF token อาจหมดหรือไม่ได้ set, GitHub raw.githubusercontent.com เป็น CORS-free ใช้ได้ทันทีโดยไม่มี dependency ภายนอก
- **UI_LAYOUT.md สร้างก่อน BIG session** — เพื่อให้วางแผนโครงการใหม่ได้โดยเห็น element ทุกตัวพร้อม ID/state/function ไม่ต้องไล่อ่าน code เอง
- **NOK แยกเป็น project ของตัวเอง ไม่ผนวกใน HORATAD/PLATFORM** — TTS มี roadmap ยาว (voice profile → cloud TTS → audio export → podcast) ถ้าฝังใน HORATAD จะ scope creep ถ้าฝังใน PLATFORM จะ couple กับ chatbot/video ที่ยังไม่เริ่ม → แยกแล้วใช้ frontend HORATAD shared ตามหลัก separation of concerns
