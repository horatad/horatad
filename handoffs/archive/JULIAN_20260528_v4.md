## WHY LOG

- **workflow_dispatch เพิ่มใน events** — Events workflow ไม่มีปุ่ม manual trigger → เพิ่ม workflow_dispatch ให้กด Run workflow ได้จาก Actions tab
- **SA×MR/SU×MR reject ไม่ใช่แค่ p-value** — แม้ p_adj < 0.05 ทุกตัว แต่ internal consistency check ชี้ว่า dist1 ในกลุ่มเดียวกันชี้ทิศทางตรงข้าม → noise ไม่ใช่ signal จริง
- **SU×MR dist5 reverse polarity** — dist5 เป็น hard aspect สำหรับ SA แต่สำหรับ MR ทำตัวเหมือน soft → อาจเพราะ natal MR position correlate กับ cohort effect ที่ซับซ้อนกว่า age proxy ตรงๆ
- **SA×SA เก็บไว้ monitor** — ต่างจาก SA×MR/SU×MR ที่ internal inconsistent; SA×SA สองกฎชี้ทิศเดียวกัน (ทั้งคู่ death) แต่ age proxy ยังไม่ชัดพอ → ต้องการ age-stratified test
