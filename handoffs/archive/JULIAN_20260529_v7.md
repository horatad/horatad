## ⭐ หลักโหราศาสตร์ที่ปีเตอร์สอน (persist — ดู JULIAN_memory.md เต็ม)

1. **นาฬิกาปลุก/spurious accuracy** — โหราศาสตร์บอกช่วงเวลา ไม่ใช่จุดเป๊ะ (คนถูกยิงไม่ตายทันที)
   → นโยบายวิจัย: beware spurious accuracy · ดาวเร็ว+ปฏิทิน Julian/Greg = ปัญหาจำกัด
2. **คนพิเศษ > คนทั่วไป** — DB บุคคลสำคัญ = จุดแข็ง · คนพิเศษลัคนาน้อย · ภพเล่าเรื่อง ดาวบอกคุณภาพ
3. **era/cohort = mundane astrology** (ของจริง) — แยก natal ออกจาก mundane ไม่ใช่ลบทิ้ง
4. **confirmatory (TALS→event) + reverse (event→planet) คู่ขนาน** — bridge เมื่อ rule base พร้อม
5. **replication = ผู้พิพากษา spurious** — ผลที่อยู่ข้าม sample = จริง · justify ด้วย out-of-sample

## DECISION FRAMEWORK — ปัจจัยเลือกงานวิจัย (6 แกน, ปีเตอร์ร่วมพัฒนา)

| ประเภท | ปัจจัย |
|---|---|
| Gate (ผ่าน/ไม่ผ่าน) | ความครบถ้วน · verifiability · policy risk |
| Benefit (max) | interest · astro value · moat |
| **Validity (ตัวที่เดิมลืม)** | **confound-resistance · statistical power** |

- interest ↔ completeness **ไม่ orthogonal** (คนดัง→บันทึกเยอะ) — แกนอิสระแท้คือ confound + power
- **Google Trends/Pageviews = สองนาฬิกา**: validity ห้ามแตะ interest · interest ใช้แค่จัดคิว+จังหวะเผยแพร่
- Wikipedia Pageviews > Google Trends (map ตรง QID + ฟรีเสถียร + ได้ event-spike feature แถม)
