## 📣 FB CONTENT PIPELINE — ทำทุก session (อ่านก่อนเลือกงาน)

> JULIAN เป็น "นักวิจัย + นักเขียน content" ให้ปีเตอร์ — ทุก session ต้องทำ 2 อย่างนี้ **ก่อนงานอื่น**:

### Step 1 — ตรวจ approved proposals → สร้าง FB caption
```
content/julian_proposals.json → หา status="approved"
→ รัน analysis จริง (ดึง valid rules จาก ml/models/julian_valid_rules.json)
→ เขียน caption ภาษาไทย + ตัวเลขจริง (lift, n, p-value)
→ บันทึก content/inbox/<timestamp>_julian_<pair>.json
→ อัปเดต proposal status → "done"
```
**ตัวอย่าง caption format ที่ดี:**
```
🔭 โหราทาส วิจัย: เมื่อดาวอาทิตย์จรผ่านดาวเสาร์นาตาล
จากข้อมูล 122,788 บุคคลสำคัญ — พบว่า ช่วงนี้มีโอกาสเสียชีวิตสูงกว่าปกติ 1.16 เท่า (n=2,823, p<0.001)
และโอกาสได้รับรางวัลต่ำกว่าปกติ 0.83 เท่า (n=2,614, p<0.001)
กฎนี้ผ่านการตรวจสอบ cross-validate ทุกยุค + ทุกประเทศ ✅
#โหราทาส #TALS #โหราศาสตร์ไทย #AstrologyResearch
```

**inbox file format:**
```json
{
  "id": "<timestamp>_julian_<pair>",
  "title": "วิจัย: <thai_pair> กับ <event>",
  "body": "<caption ภาษาไทยพร้อมโพสต์>",
  "type": "research_finding",
  "source_proposal": "<proposal_id>",
  "created_at": "YYYY-MM-DD",
  "status": "pending_review"
}
```
