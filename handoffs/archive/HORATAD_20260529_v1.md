## WHY LOG

- **Time picker แทน text input** — mobile keyboard บัง UI + error-prone manual entry → modal grid ไม่มี keyboard ปัญหา, UX เหมือน province picker
- **DST −1 ไม่ใช่ +1** — birth cert แสดงเวลา DST (clock+1 จาก standard) → ต้องลบ 1 เพื่อได้ solar time → `(hr+23)%24`
- **Foreign cities เป็น English + A-Z** — ชื่อภาษาไทยอ่านยากสำหรับดวงต่างประเทศ, A-Z filter ช่วย scan เร็วขึ้น
- **z-index 1000 > backdrop 999** — backdrop intercepted touch events → scroll/tab ใช้งานไม่ได้ทั้งหมด (root cause)
- **BGM pause เมื่อเปิดหน้าย่อย** — contact/lunar sub-page เป็น informational ไม่เหมาะกับ bgm เล่นต่อ; กลับ about-main → resume
