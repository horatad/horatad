## มุมสัมพันธ์ (Aspects)

| ชื่อ | นับราศี | กำลัง % |
|---|---|---|
| กุม | ร่วมราศีเดียวกัน | 100% |
| เล็ง | 7 ราศี (ตรงข้าม) | 80% |
| โยค | 3 หรือ 11 ราศี | 60% |
| ตรีโกณ | 5 หรือ 9 ราศี | 50% |

**กุมลัคนา** = ดาวอยู่ราศีเดียวกับลัคนา (100% ชัดเจนที่สุด)
**สัมพันธ์ถึงลัคนา** = กุม/เล็ง/โยค/ตรีโกณ รวมทั้งหมด

## Priority การแสดงผล (confirmed ch101)
กุม > เล็ง > โยค > ตรีโกณ
- ดาวกุมลัคนา → แสดงออกก่อนเสมอ แม้มีดาวอื่นสัมพันธ์ด้วย
- ดาวตรีโกณ/โยค → แสดงเมื่อให้เวลาหรือโอกาสที่เหมาะ

## Number Disambiguation ⚠️

ตัวเลขอารบิก/ไทย (1-10/๑-๑๐) = planet_id ได้ **แต่ต้องดูบริบทก่อน**

**สัญญาณว่าเป็น planet_id:**
- "ดาว N" / "ดวง N" — ตามหลังคำว่าดาว/ดวง
- "(N)" — ในวงเล็บตามหลังชื่อดาว เช่น "อาทิตย์ (๑)"
- "N กุมลัคนา" / "N อยู่ราศี" — N เป็น subject ของกริยาดาว

**ไม่ใช่ planet_id:**
- "บทที่ N" / "ข้อ N" / "ตอนที่ N" — chapter/section
- "ที่ N" / "อันดับ N" — ranking
- "ราศี N" / "ภพ N" — เลข index ราศี/ภพ

## ⭐ Subgroup Property ของตำแหน่งสัมพันธ์ลัคนา (added 2026-05-29)

ตำแหน่ง (นับจากลัคนา inclusive) ที่ถือว่าสัมพันธ์ลัคนา: **{1, 3, 5, 7, 9, 11}**

เซตนี้เป็น **subgroup** ของ ℤ/12ℤ ภายใต้การบวก mod 12:
- ผลรวมของสมาชิกสองตัวใดๆ (mod 12) ยังอยู่ในเซตเดิม
- ตัวอย่าง: 3 + 7 = 10 (ไม่อยู่ในเซต) → แต่ระยะห่างระหว่างตำแหน่ง 3 กับ 7 จากลัคนา = 7-3 = 4 (โยค) ✓

**ผลที่ตามมา (สำคัญมาก):**

> ถ้าดาว A อยู่ในตำแหน่ง a จากลัคนา (a ∈ {1,3,5,7,9,11})
> และดาว B อยู่ในตำแหน่ง b จากลัคนา (b ∈ {1,3,5,7,9,11})
> แล้ว |a - b| mod 12 จะเป็นตัวเลขคู่เสมอ ซึ่งตรงกับ aspect pattern

→ **ดาวสองดวงที่ต่างสัมพันธ์ลัคนา จะสัมพันธ์กันเองด้วยเสมอ — 100% certainty**

**นัยต่อการพยากรณ์:**
- เวลาเกิดเหตุการณ์สำคัญ: ดาวจรสัมพันธ์ลัคนา + ดาวพื้นดวงสัมพันธ์ลัคนา (Rule #2) → ทั้งสองสัมพันธ์กันเองโดยอัตโนมัติ
- ไม่ต้องตรวจ transit-natal aspect แยก — เป็นผลที่ตามมาจาก Rule #2 อัตโนมัติ
- คำถาม "ดาวจรต้องสัมพันธ์กับดาวพื้นดวงโดยตรงไหม?" → คำตอบ: ถ้า Rule #2 ทำงาน ใช่เสมอ

## ⭐ ภพเรือน = Story Domain เท่านั้น (added 2026-05-29)

หลักการสำคัญที่แยก TALS ออกจากทุกระบบ:

```
ภพเรือน  →  SUBJECT    "เรื่องนี้เกี่ยวกับอะไร?"
ดาวเคราะห์ →  QUALITY    "เรื่องนั้นจะออกมาดีหรือเลว?"
```

- ภพ 7 (ปัตนิ) บอกแค่ว่า "เรื่องนี้เกี่ยวกับคู่ครอง" ไม่ได้บอกว่าดีหรือแย่
- คุณภาพของเรื่องคู่ครอง = คุณภาพของเจ้าเรือน 7 (ดาวที่ปกครองราศีภพ 7)
- คุณภาพเจ้าเรือน = กำหนดโดย **ราศีที่ดาวสถิต** ไม่ใช่ภพ

**เปรียบเทียบ:**
- Western: Mars in house 7 = Mars nature (นักรบ/ความขัดแย้ง) merge กับ house 7 domain (คู่ครอง) → "ทะเลาะกับคู่ครอง"
- TALS: house 7 = domain คู่ครอง | เจ้าเรือน 7 อุจ/นิจ = คุณภาพชีวิตคู่ (ดาวอะไรไม่สำคัญเท่า quality ของมัน)

**ผลต่อการวิจัย:** planet quality รู้จากวันเกิด (ราศีคงที่ตลอดวัน) → quality research ทำได้โดยไม่ต้องรู้เวลาเกิด
