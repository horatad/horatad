## ธาตุ — interaction rules (ch007 verbatim)
**Source:** ch007 (2026-05-27 empirical read)

### ธาตุราศีทั้ง 12
| ธาตุ | ลักษณะ | ราศีทวาร (ต้น) | กลาง | ปลาย |
|---|---|---|---|---|
| ไฟ | เร่งรีบ ร้อนแรง | เมษ (0) | สิงห์ (4) | ธนู (8) |
| น้ำ | นุ่มนวล ปรับตัวได้ | กรกฎ (3) | พิจิก (7) | มีน (11) |
| ลม | ว่องไว | ตุล (6) | กุมภ์ (10) | มิถุน (2) |
| ดิน | หนักแน่น | มังกร (9) | พฤษภ (1) | กันย์ (5) |

**ราศีทวาร** = ต้นธาตุ (Cardinal signs) — เมษ, กรกฎ, ตุล, มังกร

### กฎ ดาวธาตุ + ราศีธาตุ (ch007 + ch022 verbatim)

⚠️ **TALS weight: "มีผลไม่มากนัก เป็นข้อสังเกตเท่านั้น"** (ch022) — ใช้เป็น modifier สุดท้าย ไม่ใช่ rule หลัก

| ดาวธาตุ + ราศีธาตุ | ผล | ปริมาณ |
|---|---|---|
| ไฟ + **ไฟ** (ต้น=เมษ) | กำลังสูงสุดทวีคูณ | สูงสุด |
| ไฟ + **ไฟ** (กลาง=สิงห์) | กำลังสูง | ต้น > กลาง > ปลาย |
| ไฟ + **ลม** | ช่วยพัดเพิ่มกำลัง (ลมโหมไฟ) | เพิ่มกำลัง (แต่น้อยกว่าไฟ+ไฟ) |
| ไฟ + **น้ำ** | ไฟตกน้ำ | อ่อนลง **เกือบครึ่ง** |
| ไฟ + **ดิน** | ไฟตกดิน | **ไม่มีผลเสียหาย** (neutral) |

**Logic ขยาย** (ตามหลักธาตุ 4):
- ธาตุเดียวกัน = สูงสุด (ต้น > กลาง > ปลาย)
- ลม + ไฟ = เพิ่มกำลัง | น้ำ + ไฟ = ลดกำลัง | ดิน + ไฟ = neutral

**ลำดับการใช้ (ch022):** วิเคราะห์คุณภาพดาว + คู่ดาว + aspect ครบก่อน → แล้วค่อยบวก modifier ธาตุราศีทีหลัง (สุดท้าย)

**ธาตุดาว (canonical):**
- ไฟ: 1 (อาทิตย์), 7 (เสาร์)
- ลม: 3 (อังคาร), 8 (ราหู)
- น้ำ: 4 (พุธ), 6 (ศุกร์)
- ดิน: 2 (จันทร์), 5 (พฤหัส)
- วิญญาณธาตุ: 9 (เกตุ) — พิเศษ, ไม่ใช่ 4 ธาตุ
- อากาศธาตุ: 0 (มฤตยู) — พิเศษ, ไม่ใช่ 4 ธาตุ

ใช้เมื่อ: finalize คุณภาพดาว (step สุดท้าย หลัง quality+pair+aspect ครบแล้ว)
อย่าสับสนกับ: คู่ธาตุ (planet+planet same element) — นี่คือ planet+sign element

## ดาวคู่ 4 ประเภท — effect rules (ch008 verbatim)
**Source:** ch008 (2026-05-27 empirical read)

| คู่ | ดาว | Effect เมื่อร่วมราศี/สัมพันธ์กัน |
|---|---|---|
| **คู่มิตร** | 1+5, 2+4, 3+6, **7+8** | ช่วยเหลือ ส่งเสริม กำลังมากขึ้นทุกด้าน |
| **คู่ธาตุ** | 1+7(ไฟ), 3+8(ลม), 4+6(น้ำ), 2+5(ดิน) | เข้ากันได้สนิท (พื้นธาตุเดียวกัน) |
| **คู่สมพล** | **1+6, 2+8, 3+5, 4+7** (memory: ๑๖ ๒๘ ๓๕ ๔๗) | ส่งเสริมกัน กำลังมากขึ้น (คล้ายมิตรแต่น้อยกว่า) |
| **คู่ศัตรู** | 1+3, 4+8, 6+7, **2+5 (เล็กสุด)** | บ่อนทำลาย อ่อนกำลังทั้งคู่ ขัดแย้ง |

**พิเศษ:**
- 7+8 (เสาร์+ราหู) = **คู่มหามิตร** — แม้ต่างฝ่ายเป็นบาปเคราะห์ แต่สัมพันธ์กัน → โชคใหญ่
- 2+5 (จันทร์+พฤหัส) = **ข้อยกเว้น** — เป็นทั้งคู่ธาตุดิน AND คู่ศัตรูเล็กสุด → ร่วมราศีให้คุณกริยามารยาท, ดีกันไม่จริง ร้ายกันก็ไม่มาก
- "คู่อื่น" (8.5) = ไม่ใช้ในการพยากรณ์ (เป็นแค่ข้อสังเกต)

ใช้เมื่อ: ดูว่าดาว 2 ดวงสัมพันธ์กันแล้วผลดีขึ้น/แย่ลงอย่างไร
อย่าสับสนกับ: คุณภาพดาวเดี่ยว — คู่ดาวเป็น modifier ของคุณภาพ ไม่ใช่คุณภาพตัวเอง

## Key Concepts

### ลัคนา (Ascendant)
- ตำแหน่งที่คำนวณจากเวลาเกิด + ที่ตั้ง
- ราศีลัคนา = ราศีที่ลัคนาอยู่
- ดาวกุมลัคนา = ดาวที่อยู่ราศีเดียวกับลัคนา

### ตนุลัคน์ (House Lord)
- ดาวเกษตรของราศีลัคนา
- เป็นตัวแทน "ตัวตนภายใน" ของเจ้าชาตา

### ตนุเศษ
- ดาวที่ส่งผลต่อตนุลัคน์
- พื้นนิสัยโดยรวม ไม่ขึ้นกับคุณภาพ/ภพ

### สามระดับ (ตนุเศษ / กุมลัคนา / ตนุลัคน์)
- ตนุเศษ = พื้นนิสัย (กว้าง ไม่เฉพาะ)
- กุมลัคนา = ลักษณะภายนอก (ชัด, หารด้วยจำนวนดาวกุม)
- ตนุลัคน์ = ตัวตนภายใน (100%, ไม่หาร)
→ สามระดับอาจเสริมหรือขัดกันในคนเดียว

### โฉลก (Auspiciousness)
- ดาวสัมพันธ์ลัคนา → ถูกโฉลก = แสดงออก ถนัด เจริญรุ่งเรือง
- ดาวไม่สัมพันธ์ลัคนา → ไม่ถูกโฉลก = ทำได้แต่ไม่เจริญ ไม่ใช่ทาง

### สมพงศ์ (Chart Compatibility) — 3 เงื่อนไข
1. ลัคนาสองคนสัมพันธ์กัน
2. ดาวกุมลัคนา/ตนุลัคน์ไม่เป็นคู่ศัตรูกัน
3. เจ้าเรือนปัตนิแต่ละคนไม่ตกอริ/มรณะ/วินาศ
→ ครบ 3 ข้อ = ผ่านเกณฑ์สมพงศ์ (ไม่ใช่ "ดีมาก" โดยอัตโนมัติ)

## ตนุเศษ — calculation + per-planet trait table (appended 2026-05-25)
**Source:** ch013 section 13.2 (workers/chapter_texts.json)

### Calculation (formula)
1. Count from ภพตนุ to ภพ where ดาวเกษตรเจ้าเรือนภพตนุ ไปปรากฏ → `a`
2. From that ภพ, look at ดาวเกษตรเจ้าเรือน → count to ภพ where THAT planet ไปปรากฏ → `b`
3. ตนุเศษ_id = `(a × b) mod 7` → planet_id 1-7 (หารลงตัว → เศษ 7 = เสาร์)
4. **Notation:** เขียน `+` หรือ `×` เหนือเลขดาวที่เป็นตนุเศษ ในแผ่นดวงชาตา

## TALS concept clarifications (user 2026-05-26)

### ฤกษ์ — ไม่ใช้ใน TALS
- โหราศาสตร์ไทยใช้ฤกษ์เพื่อ **electional** (เลือกฤกษ์ทำกิจกรรม) — **ไม่ใช่ natal/predictive**
- TALS drop ฤกษ์ เพราะ:
  - ความหมายคลุมเครือ (9 หมู่ × 3 ราศี/หมู่ → ชั่งน้ำหนักไม่ได้)
  - สำนักที่ใช้ เพื่อ "ให้มีเรื่องเล่า" — ไม่ help prediction accuracy
- ดังนั้นห้าม extract rule ที่อ้างถึงฤกษ์ในการพยากรณ์

### เรือนนอก / เรือนใน
- ไม่ใช่ "ภพ" (houses) — เป็น concept ของ **ดาว** (planets)
- **ดาว** = บอก "คุณภาพ + ปฏิสัมพันธ์ของเรื่อง"
- **ภพ** = บอกแค่ "เรื่อง" (topic/domain)
- **ตนุลัคน์ = ดาวเกษตรเจ้าเรือนลัคนา = "เรือนใน" of ตัวตน**
- "เรือนนอก" = ดาวที่ปรากฏใน chart (visible position)
- "เรือนใน" = ดาวที่เป็น ruler ของภพ (governing planet)

### อนุเกษตร — รายละเอียดเพิ่ม
- **เกิดจาก:** ดาวสลับเกษตรกัน (planets in each other's own-sign)
- **ความหมาย:** ก้าวหน้าช้าในวัยต้น · ค่อยๆ ดีขึ้นเมื่ออายุมากขึ้น
- **เหตุผล:** ตัวความเกี่ยวข้องกับ 2 เรื่องราว → ตัดสินใจไม่ได้ → ช้า
- **ต่างจาก เกษตร:** เกษตร = ให้คุณต่อเนื่อง ขึ้นเอียงตลอด · อนุเกษตร = ช้ากว่า แต่ improvement curve คล้าย

### ประเกษตร (ประ)
- **"ประ"** = **คุณภาพเสีย** (degraded quality)
- ดาว ใน ราศีตรงข้ามเกษตร = ประเกษตร = ขาดแคลน เสื่อมถอย

### ราชาโชค — Thai tradition (warning)
- เป็น **tradition ของไทย** — pure lookup per planet, **ไม่ใช่ derivable formula**
- ⚠️ **ระวังชื่อซ้ำ:** Vedic มี "Raja Yoga" แต่เป็นคนละ concept (ดาวคนละตำแหน่ง คนละเรื่อง)
- ห้าม conflate TALS ราชาโชค กับ Vedic Raja Yoga ในการ extract/wording

### TALS canonical procedure (ch020) — quick reference

**ก่อนเริ่ม 10 ขั้น — prep phase (ch020, ห้ามข้าม):**
```

② ตรวจสัมพันธ์: กุม/เล็ง/โยค/ตรีโกณ + คู่ดาว → คุณภาพเพิ่ม/ลดอย่างไร
③ จำคุณภาพดาวทุกดวงให้ได้ก่อน — ขั้นต่อไปจะใช้ตลอด
④ สอบลัคนา: ดาวเด่น vs นิสัยจริงของเจ้าชาตา (ถ้าชำนาญแล้ว)
```
→ ใช้เมื่อ: เริ่มอ่านดวงชาตาจริง ไม่ใช่แค่ทำความเข้าใจทฤษฎี
→ อย่าสับสน: prep ≠ step 1-10 — เป็นการเตรียมข้อมูลก่อนออกคำพยากรณ์

10 steps in prediction order:
1. ลัคนาราศี
2. ดาวกุมลัคนา + คุณภาพ → "ชีวิตโดยรวมจะประสบความสำเร็จแบบไหน"
3. ตนุลัคน์ + ภพ
   → สันดานแท้ภายใน = ดาวนั้น
   → ภพที่ตนุลัคน์ไปอยู่ = "ชีวิตมักเกี่ยวข้องกับเรื่องนั้น / ให้ความสำคัญเหนืออื่นใด"
   → ใช้เมื่อ: อธิบายว่าทำไมคนนี้ถึง obsess กับเรื่องใดเรื่องหนึ่ง
4. ความเข้มแข็งดวง (ตนุลัคน์ + กุมลัคนา) ← Rule #3 | บอกคร่าวๆ ไม่ต้องละเอียดในขั้นนี้
5. ตนุเศษ → นิสัยโดยรวม สั้นๆ
   ⚠️ อย่าขยายความมาก — ตนุเศษบอกภาพกว้าง ต้องดูดาวอื่นประกอบ
6. ดาว เล็ง/โยค/ตรีโกณ ลัคนา
7. ดาว NOT สัมพันธ์ลัคนา = อัจฉริยภาพภายใน
   → พูดเป็นสุดท้ายในขั้นนี้ เพราะเข้าใจยากกว่าดาวสัมพันธ์ลัคนา
   → (ch020 ระบุว่าอธิบายรายละเอียดในบทต่อไป — ยังไม่ทราบบทไหน)
8. ภพอื่น: ปัตนิ → กฎุมพะ → กัมมะ → สหัชชะ
9. อาชีพ — ดาวเด่น (NOT ภพกัมมะ)
   → ทั้งแบบ ดาวเดี่ยว และ ดาวผสม
   → (รายละเอียดในบทต่อไปของตำรา — ยังไม่ได้รับ)
10. ดาวจร

## ดาวสวมหมวก 2 ใบ — 2 บทบาทของดาวทุกดวง (ch021 verbatim)
**Source:** ch021 (2026-05-27 empirical read)

ดาวทุกดวงที่ปรากฏใน chart แสดง 2 บทบาทพร้อมกัน:

| บทบาท | ชื่อ | ลักษณะ | Rule |
|---|---|---|---|
| 1 | อัจฉริยภาพภายใน (ปัจจัยภายใน) | นามธรรม | ส่งผล **100% ทุกภพ** ดูแค่ ดาว + คุณภาพ |
| 2 | ปัจจัยภายนอก (เรือนนอกแท้) | รูปธรรม | ขึ้นกับ aspect ลัคนา (%, กุม/เล็ง/โยค/ตรีโกณ) |

### Rule อัจฉริยภาพภายใน (ch021)
> "ส่งผลถึงเจ้าชาตา 100% ทุกภพ ไม่ว่าจะอยู่ภพไหน — ดูแค่ความหมายดาว + คุณภาพ"
> **ยกเว้น:** ถ้าอยู่ **ภพ 6 (อริ), 8 (มรณะ), 12 (วินาศ)** → บวกความหมายภพต่อท้ายด้วย แต่ไม่ลด %

ใช้เมื่อ: อธิบายศักยภาพซ่อน / พรสวรรค์ที่ไม่แสดงออก / คำถาม "ทำไมดาว X ดี แต่ไม่แสดงออก"
อย่าสับสนกับ: ปัจจัยภายนอก ซึ่งต้อง aspect ลัคนาก่อนถึงส่งผล

### Rule ปัจจัยภายนอก (ch021)
- ดาวสัมพันธ์ลัคนา (กุม=100%, เล็ง=80%, โยค=60%, ตรีโกณ=50%) → ส่งผลภายนอก
- ดาวไม่สัมพันธ์ลัคนา → ส่งผลภายนอกน้อยมาก / ไม่ถูกโฉลก

### ความหมายดาว = identity คงที่ (ch021 verbatim)
ดาวจะดีมากหรือเสียมาก จะกลายเป็นดาวอื่นไม่ได้:
- จันทร์ (2) = อ่อนโยน/อ่อนหวาน — ดีก็อ่อนหวานพอดี เสียก็อ่อนเกิน แต่ไม่กลายเป็นขยัน/แข็ง
- อังคาร (3) = ขยัน/กล้า — เสียก็ขยันแต่ไม่เป็นชิ้น ไม่กลายเป็นอ่อนโยน
- พฤหัส (5) = คุณธรรม/เมตตา — เสียก็ย่อหย่อนขาดสติ แต่ไม่กลายเป็นโกง
- ราหู (8) = เล่ห์เหลี่ยม/ลุ่มหลง — ดีก็ทันคน ฉลาดทางโลก ไม่กลายเป็นมีคุณธรรม

ใช้เมื่อ: อธิบายว่าดาวเสียไม่ได้แปลว่า "ตรงข้ามกับ core meaning"
อย่าสับสนกับ: กฎลบ-ลบ=บวก ที่เปลี่ยน polarity — core meaning ยังเดิม แค่ผลดีก่อนแล้วไม่ดี หรือกลับกัน

### อัจฉริยภาพภายใน ≠ ทักษะ ≠ พรสวรรค์ (ch039 verbatim)
- **อัจฉริยภาพภายใน** = ศักยภาพที่มาแต่กำเนิด, ไม่ฝึกก็มี, ไม่ถ่ายทอดทาง DNA ก็มี
- **ทักษะ** = เกิดจากการฝึกฝน (acquired)
- **พรสวรรค์** = สามารถถ่ายทอดทางพันธุกรรมได้ (hereditary)
- คำนี้ TALS สร้างขึ้นเอง (ไม่มีในตำราโหราศาสตร์ไทยเดิม) — อ้างอิง Howard Gardner's 8 intelligences
- แสดงผ่านดาว 8 ดวง (1-8) — เกตุ (9) + มฤตยู (0) ยกเว้น (ยืนยัน ch039)

### การพิจารณาดาวครบ 2 บทบาท (ch021)
เมื่อวิเคราะห์ดาวดวงใด → ต้องพิจารณา 2 ครั้ง:
1. ในฐานะ อัจฉริยภาพภายใน: ดู ดาว + คุณภาพ (100% ทุกภพ)
2. ในฐานะ ปัจจัยภายนอก: ดู aspect ลัคนา + ความหมายดาว + ภพ

## เสียทั้งนอกและใน — double-bad house pattern (ch028 case study)

**Pattern:** เมื่อต้องการดู "ภพ X" (เช่น กฎุมพะ/ทรัพย์):
1. **นอก** = ดาวที่ลอยอยู่ในภพนั้น (เรือนนอก) → ดูคุณภาพ
2. **ใน** = เจ้าเรือน/ดาวเกษตรของราศีที่อยู่ในภพนั้น (เรือนใน) → ดูคุณภาพ

| กรณี | ผล |
|---|---|
| ดาวนอก ดี + เจ้าเรือนใน ดี | ภพนั้นดีมาก |
| ดาวนอก ดี + เจ้าเรือนใน เสีย | ดีบ้าง แต่ถูกฉุดรั้ง |
| ดาวนอก เสีย + เจ้าเรือนใน ดี | เสียแต่มีโอกาสกู้คืน |
| ดาวนอก เสีย + เจ้าเรือนใน เสีย | **"เสียทั้งนอกและใน"** — เสียหนักมาก |

**ตัวอย่าง ch028 (ลัคนาตุล):**
- ภพกฎุมพะ (house 2) = ราศีพิจิก → เจ้าเรือน = อังคาร(3)
- ดาวจันทร์(2) อยู่ภพกฎุมพะ = **เรือนนอก** นิจ → เสีย
- อังคาร(3) = **เจ้าเรือนใน** นิจ → เสีย
- → "ดาวการเงินเสียทั้งนอกและใน" — ใช้เงินเก่ง ทรัพย์สินเสียหาย

ใช้เมื่อ: พยากรณ์ภพใดก็ตาม (ไม่ใช่แค่กฎุมพะ) — check ทั้งดาวนอกและเจ้าเรือนใน

## ราหูบดบังพฤหัส — dominant malefic pattern (ch028 case study)

**Pattern:** เมื่อดาวบาป (malefic) กุมลัคนา + ดาวดีก็สัมพันธ์ลัคนาด้วย:
- ดาวบาปที่กุมลัคนา = **แรงที่สุด** (กุม priority 100%)
- ดาวดีที่เล็ง/โยค/ตรีโกณ = อ่อนกว่า → ถูก "บดบัง" โดยดาวบาปกุม
- ผล: คุณธรรมมีอยู่บ้าง แต่ไม่มีเวลาแสดงออก เพราะดาวบาปครองก่อน

**ตัวอย่าง ch028:**
- ราหู(8) อุจจาวิลาส+ราชาโชค **กุมลัคนา** → แรงมาก → ลุ่มหลง โกง
- พฤหัส(5) ราชาโชค **เล็งลัคนา** (80%) → มีคุณธรรมบ้าง แต่ราหูบดบัง
- ผล: "คุณธรรมแม้จะมีบ้าง แต่ราหู(8)แรงมาก กุมลัคนาด้วย บดบังคุณธรรม(5)จนหมดสิ้น"

Logic: กุม > เล็ง (R288 priority) — ดาวบาปกุม "ออกก่อน" ไม่เปิดโอกาสให้ดาวดีเล็งแสดง

## พื้นเพของตน — birth context modifier (ch039)

**นิยาม:** สิ่งติดตัวมาแต่กำเนิด ไม่ได้อยู่ใน chart แต่กำหนดโอกาสในชีวิต
- ฐานะผู้ให้กำเนิด (รวย/จน, ยศศักดิ์)
- ชาติตระกูล / ถิ่นกำเนิด / เชื้อชาติ
- กรรมพันธุ์ (โรค, รูปร่าง, สติปัญญา)
- พรสวรรค์ (ถ่ายทอดทาง DNA — ≠ อัจฉริยภาพภายใน)
- การอบรมสั่งสอน, ความเชื่อ, ประสบการณ์ต้น

**ทำไมสำคัญ:**
> คนเกิดวัน-เวลาเดียวกัน ดวงเหมือนกันทุกประการ — แต่ "พื้นเพ" ทำให้โอกาสต่างกันตั้งแต่ต้น
→ ดวงแข็ง + ดาวการเงินดี อาจยังจนได้ถ้า "ขาดโอกาสตั้งแต่เริ่มต้น" (ตย. จากประเทศยากจน)

**ใช้เมื่อ:** อธิบายความต่างระหว่างคนดวงเดียวกัน | อธิบายว่าทำไมพยากรณ์ accuracy ต่ำในบางกรณี
**อย่าสับสนกับ:** อัจฉริยภาพภายใน (ศักยภาพ) ≠ พรสวรรค์ (DNA) ≠ ทักษะ (ฝึก)

## ถูกโฉลก vs ไม่ถูกโฉลก — full mechanics (ch039)

**ถูกโฉลก** = ดาวสัมพันธ์ลัคนา + คุณภาพดี = ทำได้ดี + ประสบความสำเร็จ + "ตลอดเวลา"
**ไม่ถูกโฉลก** = ดาวไม่สัมพันธ์ลัคนา = ทำได้ดีในฐานะ "อัจฉริยภาพภายใน" แต่:
- ไม่ตลอดเวลา (ทำได้แต่ไม่สม่ำเสมอ)
- ถ้าแข่งขัน จะเสียเปรียบคนที่ถูกโฉลก (สายอาชีพเดียวกัน)

**ตัวอย่าง ch039 (ดาวศุกร์ ไม่สัมพันธ์ลัคนา):**
> "แม้บางคนจะแสดงออกได้ดี แทบจะตลอดเวลา แต่ในระยะยาวมักเปลี่ยนแนว หรือสู้คนอื่นไม่ได้"

**ดาวเสีย + สัมพันธ์ลัคนา (ศิลปินดาวเสีย):**
- ทำได้แต่ไม่ถูกโฉลก — กว่าจะประสบความสำเร็จต้องผ่านอุปสรรคมากมาย
- บางคน มีชื่อเสียงหลังเสียชีวิต ก็ยังถือว่า "ไม่ถูกโฉลก"

## สอบลัคนา ด้วยดาวพุธ (ch039 + ch043/044)

**เทคนิคหลัก:** ใช้ดาวพุธ(4) เป็น anchor ตรวจ lagna เพราะพุธ = การพูดจา ชัดเจนที่สุด

**คำถามสอบ:** "เจ้าชาตาเป็นคนช่างพูดหรือเปล่า?"
- ถ้าช่างพูด → พุธ(4) สัมพันธ์ลัคนา → lagna ที่คำนวณได้สอดคล้อง
- ถ้าพูดน้อย/ไม่พูด → พุธ(4) ไม่สัมพันธ์ลัคนา → อาจต้องย้าย lagna

**ข้อระวัง (ch039):**
- อาชีพที่ต้องพูด (ครู, เซลล์) อาจ override — ต้องถามว่า "พูดเองหรือพูดเพราะงานบังคับ"
- พุธตกภพมรณะ: พูดได้แต่สับสน ต้นชนปลายไม่ถูก
- กรณีหายาก: พุธกุมลัคนาแท้ กลับพูดน้อย — ต้องดูดาวอื่นประกอบและใช้เวลา

**ใช้เมื่อ:** ไม่รู้เวลาเกิดชัดเจน | สอบ lagna จากพฤติกรรมจริง

## อาชีพ — ดาวเดี่ยว + ดาวผสม (ch029 — บทอาชีพ)

### หลักทั่วไป (3 ข้อ)
1. ดาวดี + สัมพันธ์ลัคนา → ทำได้ดี + ถูกโฉลก
2. ดาวดี + ไม่สัมพันธ์ลัคนา → ทำได้ดี แต่ไม่ถูกโฉลก (เสียเปรียบคนถูกโฉลก)
   ⚠️ **ยกเว้น ภพมรณะ** — ดาวดีขนาดไหนก็ทำเป็นอาชีพนั้นไม่ได้ (เสียหายสูญสิ้น)
3. ดาวเสีย → ทำแล้วเสื่อมเสีย ไม่ว่าสัมพันธ์ลัคนาหรือไม่

| สถาปนิก | 6 + 4 + 3 |

| ตำรวจ | 3 + 8 + 4 (+5=ตำรวจดี) |
| ทหาร | 3 หลัก (ระดับสูง: +4+7+8) |
| บัญชี/ธนาคาร | 4 + 8 + 3 (⚠️ ราหูอุจ+มหาจักร+สัมพันธ์ลัคนา → ระวังโกง) |
| เชฟระดับฝีมือ | 4 + 6 + 3 + 7 |
| ปล่อยกู้ | 8 + (5 ต้องอ่อน = ทวงหนี้ไม่ได้ถ้า 5 แรง) |

**ข้อสังเกตพิเศษ:**
- 1 (อาทิตย์) แรง = รักหน้า หยิ่ง → ง้อคน ตื้อคนไม่เป็น → เหมาะงานที่ไม่ต้องง้อ
- 5 (พฤหัส) แรง = มีคุณธรรม ใจอ่อน → ทนาย/นักธุรกิจแบบเดิมจะลำบาก

## ราหู(8) พื้น vs จร — คนละทิศ (ch039+ch032)

| | ราหูพื้น | ราหูจร |
|---|---|---|
| ให้คุณ? | ✅ ด้านความฉลาดทางโลก ทันเล่ห์เหลี่ยม | ❌ **ไม่ให้คุณเลย** |
| ให้โทษ? | บางส่วน (หุ้น หวย ลุ่มหลง) | ✅ **ให้โทษเสมอ** — ลุ่มหลง, พนัน, คดีความ, ปัญหารำคาญ |
| แม้เจอคู่มิตร (เสาร์)? | N/A | ยังให้โทษ — ไม่มีข้อยกเว้น |

**ราหู(8) เสีย (นิจ/ประเกษตร) + สัมพันธ์ลัคนา (ch032):**
- หลงงมงาย เชื่อคนง่าย ไม่ทันคน ถูกหลอกง่าย
- ยับยั้งได้ด้วย: พฤหัส(5)=สติยั้งคิด + อังคาร(3)=ตรรกะ
- แต่ไม่หักล้างกัน — ดาวยับยั้งทำได้แค่ "บดบังชั่วคราว"

## เสาร์(7) — misconception + ข้อเท็จจริง (ch039)

**❌ ความเชื่อผิด (ตำราอื่น):** "เสาร์กุมลัคนา = ชีวิตลำบาก"
**✅ ข้อเท็จจริง TALS (ch039):** เสาร์ = อดทน + ทำงานหนักได้ดี + ถูกโฉลกในงานหนัก

**Pattern สำคัญ: เสาร์(7) + ราหู(8) สัมพันธ์ลัคนา (ch039 verbatim):**
> "พบว่าส่วนใหญ่มักจะเป็นคนที่ร่ำรวย เพราะสร้างฐานะได้ด้วยตนเอง (แม้ดาวการเงินจะมีคุณภาพปกติหรือไม่ดีนัก)"
→ เสาร์(7)=งานหนัก + ราหู(8)=ฉลาดทางโลก → สร้างฐานะเอง

**ราหู(8) จร — เข้าภพใด (ch039):**
- ภพปัตนิ → ปัญหาคู่/หุ้นส่วน 100% + กระทบเจ้าชาตา 80%
- ภพกัมมะ → ปัญหาหน้าที่การงาน
- สัมพันธ์ลัคนา/ตนุลัคน์ → กระทบเจ้าชาตาด้วย

## พฤหัส(5) interactions — ช่วย vs หลงดี (ch039)

**พฤหัส(5) + สัมพันธ์ลัคนา = สติยั้งคิด:**
- ยับยั้งความโมโห (อังคาร(3) แรง) — โมโหไม่นาน
- ยับยั้งการพูดผิดพลาด (พุธ(4)) — คิดก่อนพูด
- ในดาวจร: ช่วยชีวิต/ให้ราบรื่น/มีผู้ใหญ่ช่วย

**พฤหัส(5) อุจ + ราหู(8) อ่อน + สัมพันธ์ลัคนา = "หลงดี" (ch039):**
- เชื่อคนง่าย ไม่มีเล่ห์เหลี่ยม → ถูกหลอกง่าย
- ปฏิบัติธรรมหรือทำบุญจนเกินตัว
- เรียกว่า "หลงดี" — คุณธรรมดีเกิน ขาดความระวัง

**พฤหัส(5) ตนุเศษ = ไม่โกง (แต่มีขีดจำกัด):**
> "มีขีดจำกัดสูงกว่ามนุษย์ปุถุชนโดยทั่วไปเท่านั้น — ถ้ามีปัจจัยอื่นมากพอก็ยังโกงได้"

**พฤหัส(5) ไม่สัมพันธ์ลัคนา = ขาดสติยั้งคิด:**
- ผู้ใหญ่ไม่เมตตา ไม่ช่วยเหลือในยามจำเป็น
- อังคาร(3) แรง + พฤหัส ไม่สัมพันธ์ = โมโหร้าย อาจขาดสติ

## ดาวจร — หลักการ + ลำดับพิจารณา (ch033 + ch037)

### หลักพื้นฐาน
- ดาวจร = ตำแหน่งดาวปัจจุบัน vs ดาวพื้นดวง → ดูผลกระทบ
- คุณภาพดาวจร = **แค่ดีหรือเสีย** เท่านั้น (ไม่ต้องใส่ อุจ/เกษตร ฯลฯ เพราะ short-term)
- **"ทับ"** = equivalent to กุม แต่ใช้เฉพาะกับดาวจร (ดาวจรทับดาวเดิม = strongest)
- Rule #1 เสมอ: natal 80% / ดาวจร 20% (ดาวจร activate เท่านั้น ไม่สร้างใหม่)
- Accuracy benchmark: natal >80%, ดาวจร ~50% (TALS ทำได้สูงกว่านี้)

### ลำดับพิจารณาดาวจร (ch033 verbatim)
| ลำดับ | ดาว | ลักษณะ | อยู่นาน |
|---|---|---|---|
| **1st** | มฤตยู(0) | พิจารณาก่อนเพราะอยู่นานที่สุด — ตัวป่วน แปรปรวน ไม่ให้โทษถึงตายเพียงลำพัง | **7 ปี/ราศี** |
| **2nd** | เสาร์(7) | ดาวร้ายอันดับ 1 ให้โทษถึงตาย | 2.5 ปี/ราศี |
| **3rd** | อังคาร(3) | ดาวร้ายอันดับ 3 — สุขภาพ/อุบัติเหตุ โดยเฉพาะถ้าสัมพันธ์ มฤตยู(0) | 1.5 เดือน/ราศี |
| **4th** | ราหู(8) | ให้โทษเสมอ — ไม่ถึงตาย แต่พิการ/คดี/ลุ่มหลง | 1.5 ปี/ราศี |
| **5th** | พฤหัส(5) | ดาวดี ช่วยชีวิต ไม่ให้โทษเลย | 1 ปี/ราศี |
| เสริม | ดาวเจ้าเรือนการเงิน/ปัตนิ | สำคัญมากเพราะกระทบภพโดยตรง | varies |

### กฎสำคัญ (ch033)

**มฤตยู + อังคาร สัมพันธ์กัน = อุบัติเหตุ/สุขภาพแน่นอน:**
> "สัมพันธ์ถึงดาวอังคาร ไม่ว่าในดาวจรด้วยกันหรือดาวพื้น → อุบัติเหตุ/สุขภาพ"
(ถ้าสัมพันธ์ทั้งดาวเดิมและดาวจร = ไม่พลาด)

**เสาร์จร + เสาร์เสีย:**
- เสาร์ดี(อุจ/มหาจักร)จร → ให้คุณแรง แต่โทษก็แรง (ทับลัคนาถึงตายได้)
- เสาร์เสียจร → กำลังน้อย ให้คุณน้อย แต่โทษก็เบาลง

**เสาร์จร + ราหูเดิม คู่มิตร = ให้คุณแทบไม่ให้โทษ (ch033+ch039)**

**ราหูจร = ให้โทษเสมอ ไม่มีข้อยกเว้น:**
- แม้จรไปพบคู่มิตร/ธาตุ/สมพล รวมทั้ง เสาร์(7) คู่มิตร → ก็ยังให้โทษ
- ราหูจร + พุธ(ศัตรู) สัมพันธ์ = โทษทวีคูณ
- ราหูจรทับราหูเดิม + สัมพันธ์ลัคนา = โทษถึงเจ้าชาตา

**พฤหัส(5)จร + สัมพันธ์ลัคนา = เทวดาคุ้มครอง:**
> "เวลาดาวร้ายจรให้โทษรุนแรง ดูพฤหัส(5)จรว่าสัมพันธ์ลัคนาหรือไม่"
→ ถ้าสัมพันธ์ = แม้จะตาย ก็ตายราบรื่น ไม่ทุกข์ทรมาน

**ลำดับภพที่ต้องดู:**
ตนุ (สำคัญที่สุด) → กฎุมพะ (การเงิน) → ปัตนิ (คู่ครอง) → ภพอื่น (น้อยลงตามลำดับ)

### สรุป ch037
> "ดวงชาตาแข็งขนาดไหน ก็บาดเจ็บล้มตายได้ | ดาวการเงินดีขนาดไหน ก็ล้มละลายได้"
> "พยากรณ์ดาวจร: กลางๆ ไม่ร้าย/ดีจนเกิน — ให้ข้อควรระวัง + กำลังใจเท่านั้น"

## พระ vs โจร — Comparative Analysis Framework (ch031)

**หัวใจ TALS: ไม่ตีความดาวเดี่ยว → ตีความสัดส่วนระหว่างคู่ดาว**

### พฤหัส(5) = ความดี / ราหู(8) = ความชั่ว

| ดาว | ด้านใน | ปัจจัยภายนอก |
|---|---|---|
| พฤหัส(5) | คุณธรรม, เมตตา, ใจอ่อน, มองโลกแง่ดี, สติยั้งคิด, ใฝ่ธรรมะ | พระ, ครู, ผู้ใหญ่, ผู้อุปถัมภ์ |
| ราหู(8) | ลุ่มหลง, มัวเมา, เจ้าเล่ห์, ทันเล่ห์ทันเหลี่ยม, ถูกหลอก | คนโกง, นักเลง, การพนัน, หุ้น, หวย, อบายมุข |

### 5 กรณี (ประเมินจากสัดส่วน แรงต่างกันมากแค่ไหน)

| กรณี | เงื่อนไข | ผล |
|---|---|---|
| คนดีโดยแท้ | พฤหัส(5) แรงมาก + ราหู(8) อ่อนมาก | คุณธรรมสูง แต่เชื่อคนง่าย ถูกหลอกง่าย |
| คนโกงโดยแท้ | ราหู(8) แรงมาก + พฤหัส(5) อ่อนมาก | เล่ห์เหลี่ยมสูง ขาดสติ หลอกลวงได้ |
| ทั้งพระทั้งโจร | แรงพอๆกัน | ดูว่าใครแรงกว่าเล็กน้อย → ออกทางนั้น |
| คนโง่โดยแท้ | ราหู(8) **เสีย** + สัมพันธ์ลัคนา + พฤหัส(5) **เสีย** + ไม่สัมพันธ์ | งมงาย ถูกหลอกเต็มที่ กลับตัวยาก |
| คนทั่วไป | ปานกลาง ผสมกัน | เปรียบเทียบตามหลักเดียวกัน |

### มิติเวลาของคุณภาพ (สำคัญมาก)
- **เกษตร** = ยิ่งอายุมากยิ่งแรง → ผลในบั้นปลายชีวิต
- **อุจจาภิมุข** = แรงสม่ำเสมอ แรงก่อน ไม่ตก → แรงทั้งชีวิต
- **มหาจักร** = แรงเป็นพักๆ ไม่สม่ำเสมอ → บทจะโกงโกงกว่าอุจ
- **ราชาโชค/เทวีโชค** = ทันเล่ห์ทันเหลี่ยมแต่ไม่ถึงโกง (ราหู)

### กฎ: Position overcomes quality (ราหู)
> ราหู(8) **ปกติ กุมลัคนา** > ราหู(8) **อุจ ไม่กุมลัคนา**
- ราหู(8) กุมลัคนา ดาวอื่นต้านไม่อยู่ — วัยรุ่นเกเร เสียอนาคต
- ราหู(8) อุจ ตนุลัคน์ ไม่สัมพันธ์ลัคนา = ไม่น่าเป็นห่วง

### ตัวดัดแปลง (modifiers)
- **พฤหัส(5) ตนุเศษ** = คุณธรรมเป็นนิสัยติดตัว ต้านทานราหูได้แม้ราหูแรง
- **อังคาร(3) ดี + สัมพันธ์ลัคนา** = ตรรกะ ช่วยต้านความโง่งมงาย
- **อาทิตย์(1) ดี + สัมพันธ์ลัคนา** = รักเกียรติ ลดการโกงแม้ราหูแรง
- **คู่มิตร/ธาตุ/สมพล** = ยกระดับคุณภาพดาวทั้งคู่

## คนเจ้าชู้ — สูตร (ch031)

**ดาวหลัก:** ศุกร์(6) = กำหนัดความใคร่ (ไม่ใช่ความรัก)

| สูตร | ผล |
|---|---|
| ศุกร์(6) ดี + อังคาร(3) ดี + สัมพันธ์ลัคนา | เจ้าชู้ (กล้า เปิดเผย ตรงไปตรงมา) |
| ศุกร์(6) ดี + อาทิตย์(1) ดี + สัมพันธ์ลัคนา | กำหนัดสูง **แต่ไม่เจ้าชู้** (อาย รักเกียรติ) |
| + พฤหัส(5) ดี + สัมพันธ์ | เบรก → เจ้าชู้ผู้ดี ไม่ผิดลูกเมียใคร |
| + พฤหัส(5) เสีย + ไม่สัมพันธ์ | สำส่อน ผิดลูกผิดเมียใคร |

**พระสงฆ์ดีพร้อม** = พฤหัส(5) ดี + สัมพันธ์ + ศุกร์(6) เสีย + ไม่สัมพันธ์ + ไม่ร่วมอังคาร(3)/อาทิตย์(1)

**ข้อระวัง:** ศุกร์(6) เกษตร = ไม่รุนแรงถึงกำหนัดสูง | ราชาโชค/เทวีโชค = เรื่อยๆ สบายๆ

## ลัคนา / ตนุลัคน์ / อัจฉริยภาพภายใน — กลไกจริง (ch032 + discussion)

### Analogy หลัก
- **ลัคนา = สัญชาติญาณ** — impulse ดิบ ความรู้สึกทันที ตัวแสดงออก
- **ตนุลัคน์ = มันสมอง** — กระบวนการคิด ตัดสินใจ ประมวลผล — active เสมอ ไม่ต้องการ aspect
- "คนในโลกบางคนไม่ได้ทำ ไม่ได้แสดงในสิ่งที่ตนเองรู้สึกหรือคิดไว้" = ตนุลัคน์ไม่สัมพันธ์ลัคนา

### ตนุลัคน์ — สองกรณี
| สถานะ | ผล |
|---|---|
| สัมพันธ์ลัคนา | 2 ชั้น: ตัวตนภายใน + แสดงออกภายนอก (น้ำหนักตาม aspect %) |
| ไม่สัมพันธ์ลัคนา | 1 ชั้น: ตัวตนภายในเท่านั้น — คิดแบบนั้น รู้สึกแบบนั้น แต่ไม่แสดงออก |

ตนุลัคน์ ≠ 0% เมื่อไม่สัมพันธ์ — มันสมองทำงานตลอด แค่ผลไม่ออกมาภายนอก

### อัจฉริยภาพภายใน — กลไกสมบูรณ์
- **นิยาม:** ดาวพื้นที่ไม่สัมพันธ์ลัคนา = มีศักยภาพ/ความรู้นั้นอยู่ครบ 100% ในตัว แต่ไม่แสดงออกเป็นพฤติกรรมปกติ
- **ไม่ใช่ 0%** — เป็น "ศักยภาพที่ไม่ถูกดึงออกมา" ไม่ใช่ "ไม่มี"
- **แสดงออกได้** แต่ต้องมี trigger และ "ไม่ถูกโฉลก" (ผลไม่เต็ม ไม่ราบรื่น)

### Trigger — ดาวจรใหญ่
> **ดาวจร 0, 8, 7, 5, 3 ที่มาสัมพันธ์ดาวพื้นนั้น = trigger ให้อัจฉริยภาพภายในแสดงออก**

ผลขึ้นกับ:
1. คุณภาพดาวพื้น (ดี/เสีย)
2. คุณภาพดาวจรที่มากระตุ้น
3. ภพที่ดาวพื้นอยู่: อริ=ไม่ชอบทำ · วินาศ=หมดแต่ได้คืนบ้าง · **มรณะ=สาบสูญไม่ได้สิ่งตอบแทน**
4. แต่ยังไม่ถูกโฉลก = ผลไม่เต็ม ไม่ไหลตามธรรมชาติ

## ราหู(8) กรณีต่างๆ — Matrix ครบ (ch032)

### ดาวแม่สี — หลักการสำคัญ
> "ดาวแต่ละดวงคือแม่สี มีความหมายชัดเจน ไม่ซ้ำซ้อนกัน เมื่อรวมกัน = รวมความหมาย ไม่หักล้าง ไม่หักลบ"
- เสาร์(7)ช้า + อังคาร(3)ว่องไว = คิดนาน แต่ตัดสินใจแล้วทำเร็ว
- ศุกร์(6)ศิลปะ + ราหู(8)โกง = ศิลปะในการหลอกลวง / นักแสดงมืออาชีพ
- ดาวคู่ศัตรู: ลดทอนได้แต่ไม่หักล้าง — ผลัดกันแสดง หรือร่วมกันแสดง

### ราหู(8) กุมลัคนา — Matrix

| คุณภาพ | ผล | หมายเหตุ |
|---|---|---|
| เสีย (นิจ/ประเกษตร) | งมงาย เชื่อง่าย ถูกหลอก ไม่โกง (ไม่ทันคน) | ดาวอื่นทานไม่อยู่ แม้พฤหัสร่วมกุม |
| ปกติ | ลุ่มหลง เกเรต้นชีวิต ฉลาดทันคน มักไม่โกง | ดาวอื่นทานไม่อยู่เช่นกัน |
| เกษตร/ราชาโชค | เหมือนปกติ แต่ยิ่งแก่ยิ่งเจ้าเล่ห์ | ต้นชีวิตกลับตัวได้ถ้าพฤหัสแรง |
| อุจจาวิลาส/อุจจาภิมุข/อุจ/มหาจักร | คนเจ้าเล่ห์/โกงโดยแท้ ลุ่มหลงสูงแต่ไม่งมงาย | ดาวอื่นทานไม่อยู่เลย |

**กฎ Position > Quality:** ราหู(8) ปกติ กุมลัคนา > ราหู(8) อุจ ไม่กุมลัคนา

### ราหู(8) ไม่สัมพันธ์ลัคนา — Matrix

| คุณภาพ | ผล |
|---|---|
| ดี/ปกติ | อัจฉริยภาพภายใน: ฉลาดทันคน โกงเป็นแต่มักไม่โกง ถ้าโกงก็ไม่ถูกโฉลก |
| เสีย | ไม่เล่ห์เหลี่ยม โกงไม่เป็น เชื่อง่าย ถูกหลอก (อัจฉริยภาพภายใน) |
| **มหาจักร/จุลจักร** | **⚠️ กรณีพิเศษ — คาดการณ์ยาก แม้ไม่สัมพันธ์อาจลุ่มหลง/โกงได้** |

### คนราศีกุมภ์ — ตนุลัคน์ = ราหู(8)
- ตนุลัคน์ = ตัวตนภายใน ≠ ปัจจัยภายนอก
- ไม่สัมพันธ์ลัคนา = ภายในเท่านั้น
- สัมพันธ์ลัคนา = **2 ชั้น** (ตัวตนภายใน + ปัจจัยภายนอก)

### TALS ต่างจากทั่วไป
- ❌ ทั่วไป: "ราหู(8) เล็ง = พินทุบาท ดวงร้าว ดวงแตก"
- ✅ TALS: ราหู(8) ปกติ/ราชาโชค เล็งลัคนา + คู่ธาตุ/สมพลกุมลัคนา = **ดีมาก** รับเสาร์(7)จรเป็นคุณ

### ราหู(8) ที่ดีที่สุด (TALS เลือก)
> ราหู(8) ปกติ/ราชาโชค **เล็งลัคนา** + คู่ธาตุ/สมพลกุมลัคนา + พฤหัส(5)สัมพันธ์
→ ฉลาดเล่ห์เหลี่ยมแต่ไม่โกง + รับเสาร์(7)จรเป็นคุณทุกครั้ง

### 3 ตัวตรวจสอบราหู(8) เสมอ
> ดูดาวพฤหัส(5) + อาทิตย์(1) + อังคาร(3) ประกอบทุกครั้ง
- พฤหัส(5) = สติยั้งคิด คุณธรรม
- อาทิตย์(1) = รักเกียรติ ไม่ยอมเสียหน้า
- อังคาร(3) = ตรรกะ เหตุผล

### มิติเวลาคุณภาพ (สำคัญ)
- **เกษตร** = ยิ่งแก่ยิ่งแรง (บั้นปลาย)
- **อุจจาภิมุข** = แรงก่อน สม่ำเสมอตลอด ไม่ตก
- **มหาจักร** = แรงเป็นพักๆ ไม่สม่ำเสมอ บทจะโกงโกงกว่าอุจ
