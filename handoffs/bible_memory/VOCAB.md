## Key Concepts

### ลัคนา (Ascendant)
- ตำแหน่งที่คำนวณจากเวลาเกิด + ที่ตั้ง
- ราศีลัคนา = ราศีที่ลัคนาอยู่
- ดาวกุมลัคนา = ดาวที่อยู่ราศีเดียวกับลัคนา

### ตนุลัคน์ (House Lord)
- ดาวเกษตรของราศีลัคนา
- เป็นตัวแทน "ตัวตนภายใน" ของเจ้าชาตา

### ตนุเศษ
- ดาวที่ส่งผลต่อตนุลัคน์
- พื้นนิสัยโดยรวม ไม่ขึ้นกับคุณภาพ/ภพ

### สามระดับ (ตนุเศษ / กุมลัคนา / ตนุลัคน์)
- ตนุเศษ = พื้นนิสัย (กว้าง ไม่เฉพาะ)
- กุมลัคนา = ลักษณะภายนอก (ชัด, หารด้วยจำนวนดาวกุม)
- ตนุลัคน์ = ตัวตนภายใน (100%, ไม่หาร)
→ สามระดับอาจเสริมหรือขัดกันในคนเดียว

### โฉลก (Auspiciousness)
- ดาวสัมพันธ์ลัคนา → ถูกโฉลก = แสดงออก ถนัด เจริญรุ่งเรือง
- ดาวไม่สัมพันธ์ลัคนา → ไม่ถูกโฉลก = ทำได้แต่ไม่เจริญ ไม่ใช่ทาง

### สมพงศ์ (Chart Compatibility) — 3 เงื่อนไข
1. ลัคนาสองคนสัมพันธ์กัน
2. ดาวกุมลัคนา/ตนุลัคน์ไม่เป็นคู่ศัตรูกัน
3. เจ้าเรือนปัตนิแต่ละคนไม่ตกอริ/มรณะ/วินาศ
→ ครบ 3 ข้อ = ผ่านเกณฑ์สมพงศ์ (ไม่ใช่ "ดีมาก" โดยอัตโนมัติ)

## ตนุเศษ — calculation + per-planet trait table (appended 2026-05-25)
**Source:** ch013 section 13.2 (workers/chapter_texts.json)

### Calculation (formula)
1. Count from ภพตนุ to ภพ where ดาวเกษตรเจ้าเรือนภพตนุ ไปปรากฏ → `a`
2. From that ภพ, look at ดาวเกษตรเจ้าเรือน → count to ภพ where THAT planet ไปปรากฏ → `b`
3. ตนุเศษ_id = `(a × b) mod 7` → planet_id 1-7 (หารลงตัว → เศษ 7 = เสาร์)
4. **Notation:** เขียน `+` หรือ `×` เหนือเลขดาวที่เป็นตนุเศษ ในแผ่นดวงชาตา

### Per-planet personality (verbatim from ch013)
| Planet | Trait (ตนุเศษ) |
|---|---|
| 1 อาทิตย์ | ไว้ตัว รักหน้า ไม่ยอมเสียหน้า หยิ่ง |
| 2 จันทร์ | กริยามารยาทดี อ่อนโยน อ่อนหวาน |
| 3 อังคาร | ขยันขันแข็ง (ชัดเจนที่สุด) |
| 4 พุธ | ละเอียด ทำงานประณีตเรียบร้อย ชอบข้อมูล |
| 5 พฤหัส | ใจอ่อน เมตตา ขี้สงสาร ไม่โกง (ชัดเจนมาก) |
| 6 ศุกร์ | จินตนาการ เพ้อฝัน ชอบรื่นเริงบันเทิง |
| 7 เสาร์ | คิดมาก ไตร่ตรองรอบคอบมาก ๆ |
| 8 ราหู | — ไม่มีเกณฑ์เป็นตนุเศษ |
| 9 เกตุ | — ไม่มีเกณฑ์เป็นตนุเศษ |
| 0 มฤตยู | — ไม่มีเกณฑ์เป็นตนุเศษ |

**Rule:** ตนุเศษ ignore ทั้ง คุณภาพดาว และ ภพที่ดาวอยู่. เป็นพื้นนิสัยล้วน.
**Interpretation tip:** เริ่มพยากรณ์จากตนุเศษเสมอ — สั้น + ชัดเจน + ตรงประเด็น

## TALS concept clarifications (user 2026-05-26)

### ฤกษ์ — ไม่ใช้ใน TALS
- โหราศาสตร์ไทยใช้ฤกษ์เพื่อ **electional** (เลือกฤกษ์ทำกิจกรรม) — **ไม่ใช่ natal/predictive**
- TALS drop ฤกษ์ เพราะ:
  - ความหมายคลุมเครือ (9 หมู่ × 3 ราศี/หมู่ → ชั่งน้ำหนักไม่ได้)
  - สำนักที่ใช้ เพื่อ "ให้มีเรื่องเล่า" — ไม่ help prediction accuracy
- ดังนั้นห้าม extract rule ที่อ้างถึงฤกษ์ในการพยากรณ์

### เรือนนอก / เรือนใน
- ไม่ใช่ "ภพ" (houses) — เป็น concept ของ **ดาว** (planets)
- **ดาว** = บอก "คุณภาพ + ปฏิสัมพันธ์ของเรื่อง"
- **ภพ** = บอกแค่ "เรื่อง" (topic/domain)
- **ตนุลัคน์ = ดาวเกษตรเจ้าเรือนลัคนา = "เรือนใน" of ตัวตน**
- "เรือนนอก" = ดาวที่ปรากฏใน chart (visible position)
- "เรือนใน" = ดาวที่เป็น ruler ของภพ (governing planet)

### อนุเกษตร — รายละเอียดเพิ่ม
- **เกิดจาก:** ดาวสลับเกษตรกัน (planets in each other's own-sign)
- **ความหมาย:** ก้าวหน้าช้าในวัยต้น · ค่อยๆ ดีขึ้นเมื่ออายุมากขึ้น
- **เหตุผล:** ตัวความเกี่ยวข้องกับ 2 เรื่องราว → ตัดสินใจไม่ได้ → ช้า
- **ต่างจาก เกษตร:** เกษตร = ให้คุณต่อเนื่อง ขึ้นเอียงตลอด · อนุเกษตร = ช้ากว่า แต่ improvement curve คล้าย

### ประเกษตร (ประ)
- **"ประ"** = **คุณภาพเสีย** (degraded quality)
- ดาว ใน ราศีตรงข้ามเกษตร = ประเกษตร = ขาดแคลน เสื่อมถอย

### ราชาโชค — Thai tradition (warning)
- เป็น **tradition ของไทย** — pure lookup per planet, **ไม่ใช่ derivable formula**
- ⚠️ **ระวังชื่อซ้ำ:** Vedic มี "Raja Yoga" แต่เป็นคนละ concept (ดาวคนละตำแหน่ง คนละเรื่อง)
- ห้าม conflate TALS ราชาโชค กับ Vedic Raja Yoga ในการ extract/wording
