1. **Generate short token** → developers.facebook.com/tools/explorer
   - Meta App = "โหราทาส AI" → Get Page Access Token
   - permission: `pages_manage_posts`, `pages_read_engagement`, `pages_show_list`
   - Generate → เลือก Page "โหราทาส AI" → copy token (อายุ ~1 ชม.)

   ```
   FB_APP_ID=<app id> FB_APP_SECRET=<app secret> FB_SHORT_TOKEN=<short token> \
     node workers/fb_token_exchange.mjs
   ```
   → ได้ `FB_PAGE_ID` + `FB_PAGE_TOKEN` กลับมา
   (App ID/Secret: developers.facebook.com → App → Settings → Basic)

3. **ตั้ง GitHub Secrets** → repo Settings → Secrets → Actions:
   - `FB_PAGE_ID`, `FB_PAGE_TOKEN`

4. **ทดสอบ** → Actions → "FB Auto-post" → Run workflow → dry_run=true
   จากนั้นเปิด cron ให้รันจริงทุกวัน 08:30 ICT อัตโนมัติ
